# Quick Start Guide

## 🚀 Get Started in 3 Steps

### Step 1: Install Dependencies
```bash
flutter pub get
```

### Step 2: Run the App
```bash
flutter run
```

### Step 3: Enjoy!
The app will launch on your connected device or emulator.

---

## 📱 App Navigation

### 1️⃣ Splash Screen
- See the animated logo
- Click "Start" button

### 2️⃣ Login/Signup
- Enter email and password
- Click "Login" or "Create Account"
- Toggle between modes if needed

### 3️⃣ Choose Category
- Select from 10 quiz categories:
  - General Knowledge
  - Books
  - Film
  - Science & Nature
  - Computers
  - Mathematics
  - Sports
  - Geography
  - History
  - Art

### 4️⃣ Select Difficulty
- Easy (Green)
- Medium (Orange)
- Hard (Red)

### 5️⃣ Take Quiz
- Answer 10 questions
- See instant feedback
- Track your progress

### 6️⃣ View Score
- See your percentage
- Get motivational message
- Return to home

### 7️⃣ Settings (Optional)
- View quiz history
- See past scores
- Logout

## 🔧 Troubleshooting

### App won't run?
```bash
flutter clean
flutter pub get
flutter run
```

### Dependencies issue?
```bash
flutter pub upgrade
```

### Need to reset?
```bash
flutter clean
rm -rf build/
flutter pub get
```
